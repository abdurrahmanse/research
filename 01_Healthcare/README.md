Healthcare Domain

Purpose: Research projects, clinical datasets, reproducibility notes, and domain papers.

Suggested subfolders:

- `projects/` : project-specific code and READMEs
- `datasets/` : raw and processed healthcare datasets (PHI-safe practices)
- `papers/` : PDFs, bib entries, and summaries
- `notebooks/` : EDA, preprocessing, and analysis notebooks
- `experiments/` : configs, logs, metrics
- `reports/` : final reports and policy notes

Notes: ensure privacy compliance (de-identification) when storing data.
