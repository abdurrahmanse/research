% ABSTRACT
% ============================================================
\begin{abstract}
This paper presents a comprehensive analysis of plagiarism detection systems on a challenging dataset with realistic paraphrasing patterns using three distinct approaches: TF-IDF (Term Frequency-Inverse Document Frequency), BiLSTM (Bidirectional Long Short-Term Memory), and BERT (Bidirectional Encoder Representations from Transformers). We evaluate these methods on a dataset of 100+ academic text summaries with carefully controlled challenging plagiarism patterns including sophisticated paraphrasing (60-85\% word retention with reordering), semantic similarity, and lexical variation. Contrary to conventional deep learning assumptions, our experimental results demonstrate that classical statistical methods (TF-IDF) achieve superior performance with 85.51\% accuracy and exceptional ROC-AUC of 0.9863, substantially outperforming transformer-based models (BERT: 56.52\%) and bidirectional RNNs (BiLSTM: 59.42\%). TF-IDF maintains perfect recall (100\%) while achieving 79.59\% precision, ensuring all plagiarism cases are detected with minimal false accusations. This research provides practical insights for deploying plagiarism detection systems while highlighting the importance of realistic evaluation datasets and the continued relevance of classical methods. We discuss the trade-offs between model complexity, detection accuracy, and resource requirements, offering evidence-based recommendations for different deployment scenarios in academic integrity systems.
\end{abstract}
