Imaging Domain

Purpose: Medical imaging, remote sensing, microscopy, and computer-vision-focused imaging research.

Suggested subfolders:

- `projects/` : experiment folders and code
- `datasets/` : image datasets and metadata
- `models/` : trained models and checkpoints
- `notebooks/` : preprocessing and visualization
- `papers/` : domain papers and notes

Tip: store large binary datasets outside VCS and use manifest files to track dataset versions.
